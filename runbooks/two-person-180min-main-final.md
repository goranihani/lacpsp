# Two-Person 180-Minute Main/Final Runbook

Use this as a rehearsal sheet. On site, follow organizer instructions first.

## Roles

| Role | Owner | Mission |
| --- | --- | --- |
| Track A | answer lead | conclusions, issue priority, application, counterarguments, final wording |
| Track B | verification lead | AI operation, source verification, AI-usage report, format and submission safety |

## Time Plan

| Time | Track A | Track B | Locked Output |
| --- | --- | --- | --- |
| 09:00-09:30 | review official instructions | account/device smoke test | allowed-tool status |
| 10:00-10:10 | read problem and mark questions | log official instructions and facts | intake |
| 10:10-10:25 | lock direct conclusions and A issues | choose 3-5 verification targets | issue contract |
| 10:25-10:45 | start answer skeleton | verify top sources and foreign-law signals | source status table |
| 10:45-11:05 | page budget and headings | mark HOLD/DELETE risks | answer skeleton |
| 11:05-12:05 | write core application and counterarguments | run AI checks and source ledger | draft answer |
| 12:05-12:25 | refine conclusions | draft AI-usage report | report draft |
| 12:25-12:45 | judge-style attack and corrections | consistency, metadata, anonymity | submit candidate |
| 12:45-12:50 | no new content | file, format, and USB/email readiness | final file |
| 12:50-13:00 | no edits except fatal issue | submit and confirm receipt | submission evidence |

## First 25-Minute Lock

By 10:25, stop expanding and lock:

- questions
- direct conclusion for each question
- 3-5 A-level issues
- 3 official source checks
- any foreign-law signal
- page budget
- AI-usage ledger IDs

## Stop Lines

| Time | Stop |
| --- | --- |
| 10:25 | stop rereading the whole problem |
| 10:45 | stop open-ended source searching |
| 12:05 | stop broad answer expansion |
| 12:25 | stop new issues, new cases, new foreign law |
| 12:45 | stop content additions |
| 12:50 | stop editing and submit |

## 15-Minute Handoff

```text
Locked judgment:
Newly verified source:
HOLD/DELETE:
Next 15 minutes:
Stop doing:
```
